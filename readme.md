## Node Express
A basic frontend boilerplate using node and express

## Basic Structure

```javascript
|-- css
    |-- main.css
    |-- normalize.css
|-- img
|-- js
    |-- main.js
    |-- vendor
        |-- Modernizr.js
|-- views
    |-- home
        |-- index.html
|-- index.html
```

## Installation and Usage

    $ git clone https://github.com/madhums/node-express-mongoose.git
    $ cd node-express-mongoose
    $ npm install
    $ npm start
